# Real-time Chat App - WebSockets
Ett projekt byggs under modul 9 på Glimåkra folkhögskola. Syfte var att utforska hur man skapar en realtidsupplevelse på webben.

# Överblick
Applikationen är ett rita och gissa spel där en användare i taget ritar det ord som kommer upp i chatten. Andra användare kan sedan i realtid gissa och chatta med varandra. 

# TechStack
- Javascript
- HTML5 Canvas
- CSS
- WebSocket



# case-9-websocket-chat

### Gör såhär för att komma igång
Börja med att klona ner repot. 
```bash
git clone https://github.com/ComponentDig/case-9-websocket-chat.git
```

Öppna sedan terminalen och installera dependencies med:

```bash
npm install
```

Starta applikationen med:

```bash
nodemon server
```

Öppna en webbläsare och ange url 'http://localhost:8080'

Eller använd denna länk https://doodledaash.netlify.app/
